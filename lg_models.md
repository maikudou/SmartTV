For Example: LG49LF631V

When we break the model number down we will learn about what it is identifying:

**LG** — Manufacturer name  
**49** — Screen size in inches  
**L** — LED screen  
**F** — Model year 2015  
**6** — Series  
**3** — Model  
**1** — Revision  
**V** — DVB-S2/T2/C tuner
